# **App Name**: StampJoy

## Core Features:

- Restaurant Login: Allow restaurants to create profiles and manage their loyalty programs.
- Client Login: Allow clients to create profiles and track their loyalty points.
- QR Code Generation: Generate unique QR codes for each restaurant to track customer visits.
- QR Code Scanning: Scan QR codes to validate customer visits and award loyalty points.
- Loyalty Points Tracking: Track customer loyalty points and rewards progress.
- AI Reward Suggestion: Use AI to suggest appealing rewards for restaurants to offer, configured as a tool that reasons when suggestions should be included.
- Referral Code System: Enable customers to refer friends and earn bonus points.

## Style Guidelines:

- Primary color: Warm coral (#E67352) to evoke a sense of energy and excitement.
- Background color: Soft peach (#F9E8E1), a very desaturated tint of the primary, creates a welcoming, light feel.
- Accent color: Muted gold (#D4A66A), analogous to coral, to highlight key elements such as reward notifications.
- Body and headline font: 'PT Sans', a humanist sans-serif font.
- Use clean, modern icons from Font Awesome to represent different actions and categories.
- Card-based layout with rounded corners for a modern and user-friendly experience.
- Subtle animations for button interactions and reward unlocks.
